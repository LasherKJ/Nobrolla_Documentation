    Text Detection

    The Nobrolla API In Black feature analyzes video and searches for segments where video and audio would be expected, but no audio is being generated and the image is black.

    Request JSON body

    {
    "inputUri": "input-uri",
    "features":["TEXT_DETECTION"],
    "videoContext":{
        "textDetectionConfig":{
            "languagHints":"spanish"
        }
    }
}

    Options

    Attribute Type Default Description
    languageHints enum (“english”, “spanish”, “any”) “english” Language to detect.
