    Person Recognition

    The Nobrolla API Person Recognition feature analyzes video searching for a specific person

    Request JSON body

    {
    "inputUri": "input-uri",
    "features":["PERSON_RECOGNITION"],
    "videoContext":{
        "personRecognitionConfig":{
            "model":["face.jpg","face2.jpg"]
        }
    }
}

    Options

    Attribute Type Default Description
    model [“fileUri”,“fileUri”,…] [empty array] Array of image files or video files of a person. The more files provided, the more accurate the results

    Nobrolla API JSON return

    {
    "results":[
        {"titleAnnotions":[
            {
                "confidence":"0.9981",
                "segment":
                {
                    "startTimeOffset":"12.22s",
                    "endTimeOffset":"19.19s"
                }
            }
        ]},
    ]
}
